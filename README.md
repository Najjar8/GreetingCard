# 🎉 Greeting Card App

My first Android application built using **Kotlin** and **Jetpack Compose**.
This project was created as a hands-on way to learn and apply fundamental mobile development concepts in a structured and practical way.

---

## 🚀 Features

* ✨ Create and display greeting messages
* 🎨 Simple and clean UI built with Jetpack Compose
* 🌗 Supports modern UI design principles
* ⚡ Fast and lightweight
* 📱 Fully native Android application

---

## 🧠 What I Learned

This project helped me understand and apply:

* 🧩 **Composable UI design (Jetpack Compose)**
* 🔄 **State management**
* 🏗️ **Basic app architecture**
* 🎯 **Separation of concerns**
* 📦 **Project structuring for scalability**
* 🎨 **UI/UX fundamentals**

---

## 🛠️ Tech Stack

* **Language:** Kotlin
* **UI Toolkit:** Jetpack Compose
* **Architecture:** Basic MVVM principles
* **IDE:** Android Studio

## ▶️ Getting Started

### Prerequisites

* Android Studio installed
* Android device or emulator

### Run the App

1. Clone the repository:

```bash
git clone https://github.com/your-username/greeting-card-app.git
```

2. Open in Android Studio

3. Run the project on an emulator or device
